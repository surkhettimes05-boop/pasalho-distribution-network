# Phase 1 Implementation Plan

## Part 1 — App foundation
- Make the app run reliably on a standard local port.
- Ensure the main dashboard loads without crashing.

## Part 2 — Admin access
- Add a simple login screen for the admin user.
- Protect the dashboard so only the admin can access it.

## Part 3 — Product and retailer management
- Add product creation and editing.
- Add retailer creation and editing.

## Part 4 — Order entry flow
- Allow the admin to select a retailer.
- Add line items and quantities.
- Save the order as pending.

## Part 5 — Order list and status
- Show all orders in a clear list.
- Support pending and confirmed statuses.
- Make the workflow simple enough for first-time use.
